Your World, Visualized!

Let us help you measure your local space.

LocalSpace is a platform which analyzes and visualizes the impact of real estate projects on local communities based on users' input. LocalSpace is presented by Team Elevate from Cornell Tech and cooperated with Uber Elevate.

Get in Touch

How to use localspace

SET a Location

Set a location that you are interested in to build any kind of facility and measure the impact of its surrounding.

Select Feature

Select which kind of facility you want to build, select which feature you want to visualization such as noise, pollution, population etc.

visualization the impact

LocalSpace which generate the impact based on users' input using our advanced analytics algorithm. Hope the result will also impact you.
